# Style Guide

## File Naming
- Templates: `{type}-v{version}-{date}.md` (e.g., `trauma-evidence-synthesis-system-v3.0-2026-01.md`)
- Topics: kebab-case directories in outputs/topics/
- EPIC templates: descriptive with version and date

## Markdown Formatting
- Professional medical/academic tone
- Tables for comparative data, statistics, guideline summaries
- GRADE ratings: 1A, 1B, 1C, 2A, 2B, 2C, 3, 4
- Statistical reporting: 95% CI, NNT, ARR, exact p-values

## EPIC Templates
- ASCII-only (no em/en dashes)
- System-based ICU notes under 1000 chars
- UVA-compliant attestation formatting
- Billing-aware structure (CC vs E/M)

## Citations
- Every clinical claim requires PMID or DOI
- Never trust AI-generated PMIDs without verification
- Vancouver citation style
