# Product Context

## Users
- Trauma surgery attending physicians (primary)
- Surgical residents preparing presentations
- Medical educators at Level 1 Trauma Centers

## Problem
Creating evidence-based educational content and clinical documentation is time-intensive. No standardized system for generating GRADE-rated evidence syntheses, journal club analyses, or billing-compliant attestations.

## Solution
AI template system that standardizes evidence synthesis workflows and clinical documentation, producing consistent, high-quality outputs for teaching and billing.
